# Antigravity Study Tracker (Mac) 🚀

macOS용 백그라운드 학습 집중 및 딴짓 방지 프로그램입니다. 
학습 시작 시 설정된 블랙리스트 애플리케이션 및 브라우저 내 딴짓 사이트 탭들을 감시하여 강제로 종료하며, 오늘의 순수 누적 공부 시간을 Firebase Realtime Database로 연동해 관리합니다.

## ✨ 주요 기능
1. **블랙리스트 기반 프로그램 차단 🚨**
   - 스포티파이(`Spotify`) 등 유용한 음악 스트리밍 앱은 허용하며, 순수 오락 목적만 가진 지정된 블랙리스트 애플리케이션을 감지 시 운영체제 수준에서 즉각 강제 종료합니다.
2. **딴짓 웹 사이트 탭 종료 🌐**
   - Chrome, Safari, 네이버 웨일 브라우저에서 유튜브, 인스타그램, 페이스북, 틱톡, 트위터(X), 넷플릭스, 트위치 등 딴짓 사이트 접속을 실시간 탐지하여 해당 탭만 지능적으로 닫습니다.
3. **ioreg 기반 스마트 자리비움 감지 ⏸️**
   - macOS 전용 IOKit 시스템 호출을 활용하여 마우스/키보드의 비활동(유휴) 시간을 초 단위로 실시간 측정합니다. 5분 이상 움직임이 없으면 공부 타이머를 일시 정지해 꼼수를 방지합니다.
4. **Firebase 실시간 클라우드 동기화 ☁️**
   - 10초 주기로 Firebase Realtime Database에 공부 시작 상태, 오늘 누적 공부 시간(초), 목표 달성 상태를 실시간 업로드 및 동기화합니다.
5. **모던 다크 테마 GUI 🎨**
   - Tkinter를 활용한 미려한 Modern Dark Blue & Teal 테마의 UI를 제공하며, macOS 시스템 가상 메뉴바 연동으로 입력창 내 복사(Cmd+C)/붙여넣기(Cmd+V) 및 백스페이스 편집 등의 단축키를 완벽히 지원합니다.

## 🛠️ 시작하기

### 1. 패키지 설치
프로젝트 루트 디렉토리에서 아래 명령어로 필요한 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

### 2. 실행 방법
```bash
python tracker.py
```

### 3. 주요 설정
- 실행 후 화면의 **오늘 목표 (시간:분)** 입력창에 `02:30` 또는 `3:00` 과 같은 형식으로 목표 시간을 지정합니다.
- 연동할 **Firebase URL**을 입력해 클라우드와 실시간 상태를 동기화할 수 있습니다.
- 설정 값과 공부 누적 시간은 `config.json` 파일에 로컬로 자동 보관되어 프로그램 재실행 시 복원됩니다.
